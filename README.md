## ðŸ’» Core Python Projects

### 1. Conversational Agent (`basic_chatbot.py`)

This project is a simple, rule-based intelligent agent designed to process user input and maintain conversational state (memory).

* **Functionality:**
    * **Intent Recognition:** Uses keyword matching to categorize user input (e.g., "what is AI", "will AI take my job").
    * **Memory Management:** Implements persistent storage (`memory.json`) to remember the user's name across sessions, enabling personalized responses.
    * **Emotion Handling:** Attempts basic emotion detection and adjusts fallback responses based on the user's inferred emotional state (e.g., "fear," "sad").
* **Demonstrated Skills:** Basic NLP, State Management, Human-Computer Interaction logic.

### 2. Multi-Layer Perceptron Implementation (`basic_neural_network.py`)

This project implements a foundational neural network from scratch, without reliance on high-level frameworks like TensorFlow or PyTorch.

* **Architecture:** A basic Multi-Layer Perceptron (MLP) with two inputs, two hidden units, and one output.
* **Training:** Features a custom implementation of the **Backpropagation** algorithm for weight updates.
* **Key Components Implemented:**
    * **Sigmoid Activation Function:** Used for non-linear transformation between layers.
    * **Epoch Control:** Manages the number of training iterations.
    * **Learning Rate Control:** Manages the step size for weight adjustment during training.
* **Demonstrated Skills:** Core Machine Learning mechanics, Backpropagation, Fundamental mathematical basis of neural networks.
